How to use Macbook Pro

0. MAC KEYS
    ⌘  - Command
    ⇧  - Shift
    ⌃  - Control
    ⌥  - Option
    ⇪  - Capslock

1. Scroll up with 3 fingers.
    Goto Dashboard; Can add new ones

2. Go to Desktop
    Fn + F3

3. Take Screenshot by selecting a portion on the screen
    Cmd + Shift + 4

4. Hide/Show Dock
    Hold down Command and Option keys, and then press the D key

5. Finder:
    Cmd + Shift + g : Go to a folder by entering its path
    Cmd + [ - Go Backward
    Cmd + ] - Go Forward

    Cut & Paste:
        Command+C copies the files or documents, note they won’t be ‘cut’ yet
        Command+Option+V pastes the documents into the new location, cutting it from the prior locating and moving it to the new location

17. BASH Shortcuts
    source ~/.bash_profile - How to reload .bash_profile from the command line

    Ctrl + A    Go to the beginning of the line you are currently typing on
    Ctrl + E    Go to the end of the line you are currently typing on
    Ctrl + L    Clears the Screen, similar to the clear command
    Ctrl + U    Clears the line before the cursor position. If you are at the end of the line, clears the entire line.
    Ctrl + H    Same as backspace
    Ctrl + R    Let’s you search through previously used commands
    Ctrl + C    Kill whatever you are running
    Ctrl + D    Exit the current shell
    Ctrl + Z    Puts whatever you are running into a suspended background process. fg restores it.
    Ctrl + W    Delete the word before the cursor
    Ctrl + K    Clear the line after the cursor
    Ctrl + T    Swap the last two characters before the cursor
    Esc + T  Swap the last two words before the cursor
    Alt + F  Move cursor forward one word on the current line
    Alt + B  Move cursor backward one word on the current line

20. Setting up MAC:
    a. Software to be installed:
        i. Spectacle
        ii. Karabiner
    
    b. Script to take Custom Screenshot that prompts for file name and locaiton
       https://discussions.apple.com/thread/7287386?start=0&tstart=0
        - Open Automator and create a "Run AppleScript" Service.
        - Create a services and open it to get installed
        - Once Installed you can add shortcut to it (cmd + shift + 5)

    c. Karabiner Changes to change the behavior of Capslock
        i. Control_L to Control_L + When you type Control_L only, send Esc
        ii. Change Escape Key to CapsLock
        iii. Change Option_R Key to Control_R

    d. Reassigning MAC keyboard keys
        - Open Keyboard Preference and Change Modifier Keys
            - Change Caps Lock Key to Control
        - Add the following text replacement words
            (cmd) ⌘
            (shift) ⇧
            (opt) ⌥
            (ctrl) ⌃

    e. Display date along with Clock
        - Open Data and time preferences -> Clock -> Show data & Use a 24 hour format

    f. Firefox Load Speed dial home page on New Tab
        - Tab Mix Plus addon: Events -> Load on New Tab: Home Page (chrome://speeddial/content/speeddial.xul)
    
    g. Install Macvim
       This has clipboard support
       sudo chown -R $USER /usr/eocal
       brew install macvim --with-override-system-vim
       alias vim='mvim -v'
